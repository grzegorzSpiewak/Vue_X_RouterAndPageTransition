# vue-cli
Unleash Quest: Vue.js Week #4: The UI and Animations Week (with vue-router)

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
